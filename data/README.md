## Overview

This directory contains the synthetic dataset used for all demonstrations in this repository. All data is synthetic, generated by [`src/generate_synthetic_data.py`](../src/generate_synthetic_data.py).

## `synthetic_doggraph_sample.csv`

### Dataset Summary

| Property | Value |
|----------|-------|
| Dogs | 100 synthetic individuals |
| Time range | January 1 – December 31, 2025 |
| Records | 36,500 (365 days × 100 dogs) |
| Missing data rate | ~7.2% (realistic sensor dropout + context disruptions) |
| Dogs with simulated behavioral drift | ~25% |

### Schema

| Column | Type | Description |
|--------|------|-------------|
| `dog_id` | string | Unique dog identifier (e.g. `DOG_0001`) |
| `date` | date | YYYY-MM-DD |
| `month` | int | Month number (1–12) |
| `breed` | string | One of 5 breeds (see below) |
| `age_group` | string | `young_adult`, `adult`, or `senior` |
| `has_drift` | bool | Whether this dog has simulated behavioral drift |
| `drift_active` | bool | Whether drift is active on this date |
| `context` | string | Behavioral context (see below) |
| `data_missing` | bool | Whether this day's sensor data is missing |
| `missingness_type` | string | `systemic`, `sensor_dropout`, or null |
| `sleep_hours` | float | Daily sleep duration (hours). NaN if missing |
| `activity_minutes` | float | Daily activity time (minutes). NaN if missing |
| `social_score` | float | Social interaction score (0–10). NaN if missing |
| `nocturnal_restlessness` | float | Nocturnal restlessness score (0–10). NaN if missing |
| `vocalization_events` | float | Vocalization count. NaN if missing |

### Breeds Included

Breed parameters are **illustrative estimates**, not validated population norms.

| Breed | Sleep mean | Activity mean |
|-------|-----------|--------------|
| Labrador Retriever | 12.5 h | 85 min |
| Border Collie | 11.0 h | 110 min |
| Golden Retriever | 13.0 h | 75 min |
| Beagle | 12.0 h | 70 min |
| French Bulldog | 14.0 h | 45 min |

### Context Labels

| Context | Description | Missing rate |
|---------|-------------|-------------|
| `normal` | Standard home environment | ~4% |
| `travel` | Dog traveling with owner | ~35% |
| `boarding` | Dog at boarding facility | ~35% |
| `owner_absent` | Owner away from home | ~15% |
| `illness_minor` | Minor illness episode | ~4% |
| `new_environment` | New home/environment | ~4% |

### Missingness Taxonomy

Following the Barkley Missing Data Paradox framework:

- **Systemic absence** — missing due to travel or boarding (device not worn or no connectivity)
- **Sensor dropout** — random technical failure
- **Informative absence** — extended periods without anomaly signals, interpretable as a stability signal in longitudinal monitoring (this is a conceptual category, not directly encoded in the raw CSV but explored in [Notebook 03](../notebooks/03_missing_data_as_behavioral_signal.ipynb))

## Limitations

- This is **synthetic data**. It was generated by sampling from illustrative breed-level distributions, with simulated behavioral drift injected for ~25% of dogs.
- Drift signatures in the data are **illustrative**, not based on observed real-world behavioral progression patterns.
- Breed-level parameters are **illustrative estimates** drawn from general literature ranges, not calibrated population norms.
- The synthetic generation model makes simplifying assumptions that do not capture the full complexity of real behavioral data.

For a complete discussion of limitations, see [`docs/ethics_and_limitations.md`](../docs/ethics_and_limitations.md).
